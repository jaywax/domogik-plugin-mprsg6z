===================
Plugin mprsg6z
===================

Purpose
=======

Domogik plugin for Monoprice Amp Multizone MPR-6ZHMAUT (https://www.monoprice.com/product?p_id=10761)

* This is a no xPL plugin, sensor value are update by **0mq messages**
* The plugin use rs232 protocol for communication with 1 to 3 Amp connected with 18-pin ribbon cable.
